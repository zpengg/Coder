先看一段代码

```js
<body>
<script>
    function a() {
        /* 我就是构造函数 */
    }
 
    var b = new a();
 
</script>
 
</body>
```

`a` 是一个函数
`a.prototype` 称为 `a`函数的原型对象
![](http://zpengg.oss-cn-shenzhen.aliyuncs.com/img/160628888131b464.png)
`a.prototype.constructor` 指向 `a`

通过`new a()` 的到的对象`b`
`b.__proto__` 同样的 也是指向 `a.prototype` 这个对象
