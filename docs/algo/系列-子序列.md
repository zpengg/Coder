 ## 套路

### 一维dp
#### 特点
一个字符串

第一种思路模板是一个一维的 dp 数组：
```java
int n = array.length;
int[] dp = new int[n];

for (int i = 1; i < n; i++) {
    for (int j = 0; j < i; j++) {
        dp[i] = 最值(dp[i], dp[j] + ...) // 参数个数与i有关
    }
}
```
这种情况也常可用[[贪心算法]]
