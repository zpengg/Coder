# 大数
## 1e9+7 取余数
double 可以使用 1e9+7 的写法
最后建议强转回来
return (int) (ans % MOD); 
其实不止1e9+7，还有1e9+9和998244353。这三个数都是一个质数，同时小于2的30次方 。
这种数的奇妙之处在于：

如果原数是Int，那么所有模过的数之间的加法操作必然不会溢出int
如果原数是long,那么所有模过的数两两相乘必然不会溢出long
