# 319 灯泡开关
## 时间
[[2021-01-18]]
## 题目
[NO.319](link)
初始时有 n 个灯泡关闭。

第 1 轮，你打开所有的灯泡。 第 2 轮，每两个灯泡你关闭一次。 第 3 轮，每三个灯泡切换一次开关（如果关闭则开启，如果开启则关闭）。

第 i 轮，每 i 个灯泡切换一次开关。 对于第 n 轮，你只切换最后一个灯泡的开关。

找出 n 轮后有多少个亮着的灯泡。
## 相关概念
奇偶数
平方数

## 思路
亮着的按了奇数次
偶数次负负得正 还是黑的

不等的因子: 成对 2*3 = 3*2
相等的因子：**平方数** 两因子相同，对应题意，只会被按一次 
平方数有奇数个因子

## code
return (int)Math.sqrt(n);
