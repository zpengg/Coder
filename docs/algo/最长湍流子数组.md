## 思路
窗口宽度：r-l+1

## code
```java

class Solution {
    public int maxTurbulenceSize(int[] arr) {
        int n = arr.length;
        int r = 0;int l = 0;
        int max = 1;

        while(r < n-1){
            if(l== r){
                while(l<n-1 && arr[l] == arr[l+1]){
                    l++;
                }
                r = Math.min(n-1, l+1);
            }else{
                if (arr[r] > arr[r-1] && arr[r] >arr[r+1]) {
                    r++;
                }else if(arr[r] < arr[r-1] && arr[r] <arr[r+1]){
                    r++;
                }else{
                    l = r;
                }
            }
            max = Math.max(r-l+1, max);
        }
        return max;
    }
}

```
