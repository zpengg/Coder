## 背景

## 目标
穷举过程种记录满足条件的方法数 

## 问题关键词
 - 穷举
 - 方法数
 - 路径数
 - 多少种可能的结果

## 概念
[[树遍历]]

## 思路

1. 遍历路径结束条件
2. 符合状态条件
3. 位置等状态
4. 当前状态可选的操作
5. 当前状态如何更新操作

## 优化
备忘录 剪枝

## 套路 
```
result = []
def backtrack(目前状态, 操作[]):
    if （状态）满足结束条件:
        result.add(路径)
        return

    for 操作 in 操作[]:
        新状态 = f(目前状态, 操作 )
        backtrack(目前位置, 操作[]，新状态值)
```
