# 红黑树
先了解下红黑树的基本性质
 - 节点分红/黑色 boolean red
 - 根节点 黑色
```
root must black:
[black]
    |
[black]
```
 - 红子节点的子节点黑色。
```
red son must black;
root:[black]
    ||
[RED][black]
  |
[black] 
```
 - 每个叶节点（NIL节点 无数据） 是黑色
```
null/NIL is black
```
 - root到NIL的黑色节点数目相同
```
path black cnt equal

   [black]
    |  |
[RED]   [black]
  |     cntBlk= 2
[black] 
cntBlk= 2
```
